# WMS API

Warehouse Management System — .NET 8 Web API

## Tech Stack
- .NET 8 Web API
- Entity Framework Core
- SQL Server

## Features
- Flow 1: Goods Receiving
- Flow 2: Unload & Basket Loading
- Offline Support Queue
- Cancel Approval System

## Setup
1. Clone repo
2. Copy appsettings.Development.json
3. Update connection string
4. dotnet run
